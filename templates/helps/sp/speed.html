Los ajustes de velocidad solo se usan para el modo de control directo, pero para los usuarios de RAMPS y similares también están accesibles en los cuadros de código de GCODE.

Tipos de Velocidades

Mínima - Configuración
Se usa cuando las impresiones se sumergen en la resina.
Inicio - Configuración
Se usa para la aceleración. No debería ser demasiado alto ya que es posible perder pasos.
Máxima - Configuración
Se usa como velocidad máxima cuando la impresora no está imprimiendo.
Sección Lenta - Perfil
Velocidad que se usa cuando la distancia de la plataforma a la superficie del tanque es menor que este valor.
Máxima - Perfil
Velocidad Máxima durante la Impresión
Dinámica - Perfil
Anular todas las velocidades excepto la velocidad de inicio durante la impresión.
