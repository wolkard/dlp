Versión de Raspberry Pi / Linux

Los puertos USB/Serie normalmente escuchan una de estas direcciones:
 /dev/ttyACM0
 /dev/ttyACA0
 /dev/ttyUSB0

Para averiguar la dirección correcta, puedes conectarte a la Raspberry a través del protocolo SSH usando herramientas como Putty. Ejecuta el siguiente comando y anota las rutas disponibles.

ls /dev/tty*

Conecta tu dispositivo a la Raspberry Pi y vuelve a escribir el comando anterior. La ruta adicional pertenece a tu dispositivo.


Versión de Windows

Introduce la dirección del puerto COM de la RAMPS.
Pasos para encontrar el puerto COM correcto en Windows:
  1. Abre el cuadro de diálogo "Ejecutar" presionando y manteniendo presionada la tecla de Windows, luego presiona la tecla R ("Ejecutar").
  2. Escribe devmgmt.msc y presiona enter.
  3. Consulta la sección de Puertos en el Administrador de Dispositivos.
