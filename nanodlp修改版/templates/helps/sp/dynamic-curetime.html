El resultado de las ecuaciones se usará como un valor para el tiempo de curado. La unidad de tiempo de curado es el Segundo.

Al usar esta función, se ignorarán los tiempos de curado, incluido el tiempo de curado de las Capas de Fijación.

Si el valor del tiempo de curado fuese negativo, el resultado se descartará y se utilizará el tiempo de curado normal.
