Solución de Problemas

Problemas de Imagen

Sin Imagen

  • Raspberry Pi: Modo HDMI incorrecto, edita el archivo config.txt.
  • Versión de Escritorio: La Pantalla no se inicia desde el Panel de Control.
  • Número de Pantalla incorrecto.
  • El negro o un color próximo al negro ha sido elegido como color de renderizado de capa.


Imagen volteada

  • Raspberry Pi: edita el archivo config.txt.
    • 90 degrees - display_rotate=1
    • 180 degrees - display_rotate=2
    • 270 degrees - display_rotate=3
    • horizontal flip - display_rotate=0x10000
    • vertical flip - display_rotate=0x20000
  • Proyector: puedes corregir el problema del volteo de la imagen usando las opciones del proyector.


Problemas de Impresión

Problema en la Dimensión X/Y

  • La Resolución del Proyector en la página de configuración es incorrecta.
  • La Resolución X/Y en la página de configuración es incorrecta.
  • El valor para Anular la Resolución X/Y en la página del perfil es incorrecto.
  • El valor para Modificar la Resolución X/Y en la página del perfil es incorrecto.


Problemas de Movimiento

Distancia de Movimiento Incorrecta

  • RAMPS: Modo de posicionamiento incorrecto en la página de configuración (Relativo/Absoluto)
  • Control Directo: Los detalles del Eje-Z en la página de configuración son incorrectos.
  • ontrol Directo: El movimiento es demasiado rápido, los pasos se omiten. Cambia la velocidad de movimiento.

Dirección de Movimiento Incorecta

  • Cambia la dirección del eje desde la página de configuración


Problemas con la Cortadora

Partes perdidas o adicionales en una capa

  • El archivo fuente STL tiene problemas, compártelo para que podamos analizar si podemos resolver el problema o no.


Problemas NanoDLP

El programa se bloquea o se reinicia

  • Ve a la Página de Configuración/Herramientas y presiona el botón de depuración para guardar los archivos de registro relativos al bloqueo, y compártelos con nosotros.
  • En algunos casos, los bloqueos están relacionados con problemas de alimentación. Asegúrate de que la fuente de alimentación sea fiable y lo suficientemente potente.
